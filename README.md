
## Getting Started

First, run the development server:

```bash
yarn dev  - client
yarn server  - server
```

Open [http://localhost:9999](http://localhost:9999) with your browser to see the result.

## Technologies used

 ### Client
 * NextJS
 * typescript
 * @reduxjs/toolkit
 * react-redux
 * next-redux-wrapper
 * @mui/material
 * socket.io-client
 ### Server
 * express
 * express-async-errors
 * jsonwebtoken
 * multer
 * socket.io
 * postgres
 * sequelize
